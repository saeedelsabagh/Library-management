# Library Management System

A desktop application developed using Java and MySQL for managing and selling books in libraries.

The system helps organize books by educational stage and subject, while allowing the librarian to manage books, quantities, and prices easily.

---

## Features

- Login System for library access
- Add new books
- Update existing books
- Delete books
- Organize books by:
  - Educational Stage
  - Subject
- Display books in tables
- Save and print book data
- Simple and user-friendly interface

---

## Technologies Used

- Java
- Java Swing
- MySQL
- JDBC
- NetBeans IDE
